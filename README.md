靠北工程師API
==============

### Development
```bash
# Composer installations
$ cd your-folder
$ composer install

# Generate new app key
$ php artisan key:generate

# Install dependencies listed in package.json
$ yarn install

# Start with front-end develope
$ gulp

# Run local server
$ php artisan serve
```

### Options
```bash
# Enable async queue driver
$ php artisan migrate
```

### Troubleshooting
```bash
# Site loading very slow or page content not automatic reload
$ php artisan dump-autoload
```
